#Student Performance Analysis

## What This Code Does
1. **Loads Data:** Creates a small dataset of 15 students with their study hours, attendance, and exam scores.  
2. **Cleans Data:** Finds a missing value in the "Study Hours" column and fills it with the class average.  
3. **Calculates Stats:** Uses summary statistics to find the average scores and attendance.  
4. **Visualizes Data:** Creates three charts:  
   - A Bar Chart showing everyone's individual score.  
   - A Scatter Plot comparing study hours to exam scores.  
   - A Pie Chart showing the gender breakdown of the class.  

## Key Insights
* **Studying Pays Off:** The scatter plot shows that students who studied more hours received higher exam scores.  
* **Attendance Matters:** The student with the lowest attendance (40%) also received the lowest score, proving that just showing up is half the battle.  
* **Class Demographics:** The pie chart indicates the class is almost evenly split between male and female students, meaning the data isn't biased.
